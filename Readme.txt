To run our project (EShopSpot), the following things are needed:

1. You should have eclipse for java developers installed on your sytem.
https://www.youtube.com/watch?v=OBuOEYjj9kc

2. In eclipse, you need to have a plugin of JavaFx.
3. SceneBuilder for JavaFx.
https://www.youtube.com/watch?v=Dbb69NiQHso&t=574s

4. For database connectivity:
	> Mysql Workbench should be set up.
https://dev.mysql.com/downloads/installer/
https://dev.mysql.com/downloads/connector/j/

5. Necessay Jar files 
	> Mysql
	> Javafx
   The user libraries should be added accordingly.

Follow the links for better understanding.
